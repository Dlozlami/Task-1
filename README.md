## Run Locally

Clone the project

```bash
  git clone https://github.com/Dlozlami/Task-1.git
```

Go to the project directory

```bash
  cd .\sneakerx\
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
